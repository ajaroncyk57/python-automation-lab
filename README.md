# Python Automation Lab

A collection of practical Python scripts, small applications, finance calculators, workflow automation tools, and archived early programming exercises.

This repository includes both polished projects and early learning exercises. The goal is to show progression from foundational Python concepts to practical business and analytics tools.

| Project | Description | Status |
|---|---|---|

| Restaurant POS Sales Reporting | Small point-of-sale (POS) and Sales Reporting application inspired by an early Python project. | Rebuild Planned |
| File & Data Utilities | Practical scripts for cleaning, organizing, and automating repetitive data workflows. | Planned |
