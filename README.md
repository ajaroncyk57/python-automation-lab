# Python Automation Lab

A collection of practical Python scripts, small applications, finance calculators, workflow automation tools, and archived early programming exercises.

This repository includes both polished projects and early learning exercises. The goal is to show progression from foundational Python concepts to practical business and analytics tools.

## Featured Projects

| Project | Description | Status |
|---|---|---|
| Capital Budgeting Investment Analysis | Python-based Investment Analysis tool for calculating NPV, IRR, ROI, Payback Period, Profitability Index, and project-level finance metrics. | Initial Rebuild Complete |
| Restaurant POS Sales Reporting | Command-line point-of-sale (POS) and sales reporting tool for capturing orders, generating receipts, and summarizing item/category sales performance. | Initial Rebuild Complete |
| File & Data Utilities | Practical scripts for cleaning, organizing, and automating repetitive data workflows. | Planned |

## Archive

The `archive/intro-python-exercises` folder contains early Python learning exercises and is not intended to represent current production-level work.
