# Capital Budgeting Investment Analysis

A Python-based Investment Analysis Tool for evaluating Capital Budgeting Projects using core finance metrics, including Net Present Value (NPV), Internal Rate of Return (IRR), and Return on Investment (ROI).

## Project Status

This project is currently a rebuild candidate based on an early Python script. The current version demonstrates the original concept and menu-driven workflow, while a future version will improve structure, usability, documentation, and testing.

## Business Problem

Capital budgeting decisions require comparing upfront investment costs against expected future cash flows. This project evaluates whether a proposed investment is financially attractive based on user-provided assumptions.

## Current Features

- Collects project assumptions from user input.
- Calculates Net Present Value (NPV).
- Estimates Internal Rate of Return (IRR).
- Calculates Return on Investment (ROI).
- Generates a basic investment analysis report.
